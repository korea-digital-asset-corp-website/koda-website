# KODA

## Installing / Getting started

```shell
npm install
```

## Developing

```shell
npm run dev
```

### Built With

```shell
npm run build
```

### Prerequisites

node version 20.16.0

### Deploying / Publishing

- master 브랜치에 머지시 자동으로 https://kodax.com/ 로 배포됩니다.
